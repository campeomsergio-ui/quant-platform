# quant-platform

Research-grade autonomous quant platform grounded in three frozen JSON source-of-truth files:
- `configs/strategy_spec.json`
- `configs/experiment_plan.json`
- `configs/repo_blueprint.json`

Status markers:
- implemented
- scaffold_only
- placeholder
- speculative_extension

Current usable CLI paths:
- `spec-check`
- `run-experiment`
- `run-baseline-local`

Local sample-data helper:
- `quant_platform.sample_data.write_sample_daily_equities_dataset(root)`
